# Local Chat: PyQt6 Client + Flask Server

Локальный чат с десктопным клиентом на PyQt6 и сервером на Flask. Позволяет отправлять и получать сообщения в локальной сети через HTTP‑запросы.

## Функциональность

* Подключение к серверу по указанному хосту и порту.
* Отправка сообщений с указанием ника (по умолчанию — `Aztfast`).
* Получение истории сообщений с сервера.
* Отображение статуса подключения (подключено/не подключено).
* Визуальные уведомления о статусе (цвет текста: красный/зелёный).
* Очистка поля ввода после успешной отправки сообщения.
* Обработка ошибок сети и ответов сервера.

## Технологии

* **Flask** — веб‑сервер для хранения и раздачи истории сообщений.
* **PyQt6** — десктопный графический интерфейс клиента.
* **requests** — выполнение HTTP‑запросов от клиента к серверу.
* **JSON** — формат обмена данными между клиентом и сервером.

## Установка и запуск

### Требования

* Python 3.8+
* pip (менеджер пакетов Python)
